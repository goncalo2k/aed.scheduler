@listStudents Lists all the students in the set, correctly filtered, according to studentFilters. Complexity: O(n²) or O(n)?

Parameters
students- Set of students read from the files.
op1- Option 1 - manages how the students will be displayed: either by name or by number.
op2- Option 2 - manages the order of the students: either ascending or descending.
op3- Option 3 - manages whether the classes of each student will be displayed or not.
op4- Option 4 - manages whether all the students will be displayed or just a part of them.
op5- Option 5 - manages whether all the students will be displayed or just students enrolled in more than n courses or less than n courses.
min- Lower bound of the interval for option 4.
max- Upper bound of the interval for option 4.
n- Number of classes for option 5.
